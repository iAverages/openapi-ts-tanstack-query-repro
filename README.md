# openapi-ts-tanstack-query-repro 

base created using with the solid template
```bash
pnpm create vite
```

create the client
```bash
pnpx tsx openapi-ts.ts
```

check tsc with and without declarations enabled
```bash
pnpm tsc -p tsconfig.declaration.json
pnpm tsc -p tsconfig.no-declaration.json
```
